What is Node?

Node is a Multipurpose Discord Bot. Developed by discord user THE ASSASSIN#0128. Node can be used to manage discord servers, moderate the servers, manage server members, manage events, reaction roles, xp roles and more.

Why choose Node?

Node is free to use discord dot. Can be used by anyone in any server.

1. Invite the bot to your server.
2. Head to the bot's website.
3. Start configuring the bot.
4. It's easy and simple.
